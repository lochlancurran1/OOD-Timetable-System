# OOD-Timetable-System
Java MVC CLI UL timetable system built in a team of 3. Built with an object-oriented design and clean separation of Model/View/Controller.

How to run:
In VS Code, open the project folder. Open src/Main/Main.java. Beside public static void main(String[] args), click run.
